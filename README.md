# üß™ Usage

### **Run an audit**

Inside any JS/TS project:

``` sh
origin audit
```

This generates: - **Console Report** (Lite Mode) -
**docs/ai/bridge_summary.md** -
**docs/audit_history/audit_TIMESTAMP.json**

These files are safe for AI ingestion (IDE assistants, docs, copilots,
etc.)
